javascript的一些知识点

isNaN( )

用来判断一个变量是否为非数字的类型，x是数字，返回false；x不是数字，返回true。

length

获取字符串的长度，返回数字(number类型)。

字符串拼接

字符串拼接用 ‘+’，方式为：字符串 + 任何类型 = 新字符串

注意，拼接前，会把 非字符串类型→字符串类型，再进行拼接。

但如果按照下述写法 ( = + ) ，结果则是number类型

typeof 和 constructor

用来查看变量类型，如果对象是 JavaScript Array 或 JavaScript Date ，我们就无法通过typeof来判断他们的类型，因为都是返回 object。

constructor 属性返回 JavaScript变量的构造函数

短路运算

短路运算的原理：当有多个表达式（值）时,左边的表达式值可以确定结果时,就不再继续运算右边的表达式的值

  • 逻辑与

    语法： 表达式1 && 表达式2
    如果第一个表达式的值为真，则返回表达式2
    如果第一个表达式的值为假，则返回表达式1

  • 逻辑或

    语法： 表达式1 || 表达式2
    如果第一个表达式的值为真，则返回表达式1
    如果第一个表达式的值为假，则返回表达式2

switch 和 if else 的区别

  • switch更适用于有确定值的选择，而 if else 更适用于进行范围判断的内容。
  • switch 的执行效率更高，原因在于：switch 语句在运行时，首先会生成一个“跳转表”来指示实际的case 分支的地址，而这个“跳转表”的索引号与 swtich 中的 case 值是全等的，这样的话，switch 就不用像 if else 那样，遍历所有的条件，直至找到正确条件，而仅仅只需要访问对应索引号的表项就可以到达定位分支的目的。

数组元素

  • 通过 “数组名[索引]“ 的形式来获取数组元素

  • 通过 for 循环索引遍历数组中的每一项

  • “数组名.length“ 可以访问数组元素的数量

  • 可以通过修改 length 来实现数组扩容，length 属性是可读写的，多余空间相当于声明了变量但没有赋值，默认值是undefined

  • 可以通过修改索引的方式追加数组元素，最常用的一种方式

函数都是有返回值的

  • 如果有return 则返回 return 后面的值

  • 如果没有return 则返回 undefined

arguments 对象

当我们不确定有多少个参数传递的时候，可以用 arguments 来获取。

所有函数都内置了一个 arguments 对象，arguments 对象中存储了传递的所有实参。

arguments展示形式是一个伪数组，因此可以进行遍历。伪数组具有以下特点：

  1. 具有 length 属性
  2. 按索引方式储存数据
  3. 不具有数组的 push , pop 等方法

作用域以及 var, let, const

  1. 块作用域 { }

     JS中作用域有：全局作用域、函数作用域。没有块作用域的概念。

    ​ ECMAScript 6(简称ES6) 中新增了块级作用域，使用 let 声明的变量只能在块级作用域里访问，有“暂时性死区”的特性（也就是说声明前不可用）。

    ​ 块作用域由 { } 包括，if 语句和 for 语句里面的 { } 也属于块作用域

  2. var、let、const 的区别

    ​ var 声明的变量属于函数作用域，let 和 const 声明的变量属于块级作用域。

    ​ var 存在变量提升现象，而 let 和 const 没有此类现象。

    ​ var 变量可以重复声明，而在同一个块级作用域，let 变量不能重新声明，const 变量不能修改。

    ​ var定义的变量，没有块的概念，可以跨块访问, 不能跨函数访问。

    ​ let定义的变量，只能在块作用域里访问，不能跨块访问，也不能跨函数访问。

    ​ const用来定义常量，使用时必须初始化(即必须赋值)，只能在块作用域里访问，而且不能修改。

全局变量 和 局部变量

  • 全局变量：在任何一个地方都可以使用，只有在浏览器关闭时才会被销毁，因此比较占内存
  • 局部变量：只在函数内部使用，当其所在的代码块被执行时，会被初始化；当代码块运行结束后，就会被销毁，因此更节省内存空间

预解析

JavaScript 解析器在运行 JavaScript 代码的时候分为两步：预解析和代码执行（从上到下执行JS语句）。

预解析也叫做变量、函数提升。在当前作用域下，JS 代码执行之前，浏览器会默认把带有 var 和 function 声明的变量在内存中进行提前声明或者定义。

变量提升： 变量的声明会被提升到当前作用域的最上面，变量的赋值不会提升。

函数提升： 函数的声明会被提升到当前作用域的最上面，但是不会调用函数。

对象

在 JavaScript 中，对象是一组无序的相关属性和方法的集合

对象里面的属性调用 : 对象.属性名 ，这个小点 . 就理解为“ 的 ”

对象里面属性的另一种调用方式 : 对象[‘属性名’]，注意方括号里面的属性必须加引号，我们后面会用

对象里面的方法调用：对象.方法名() ，注意这个方法名字后面一定加括号

创建对象的三种方式：

  • 利用字面量创建对象，{ } 里面采取键值对的形式表示

  • 利用 new Object() 创建对象，{ } 里面采取 对象.属性 = 值 的形式表示

  • 利用构造函数创建对象。 函数内的属性和方法前面需要添加 this ，表示当前对象的属性和方法。构造函数中不需要 return 返回结果。必须用 new 来调用构造函数。

    构造函数 ：是一种特殊的函数，主要用来初始化对象，即为对象成员变量赋初始值。我们可以把对象中一些公共的属性和方法抽取出来，然后封装到这个函数里面。

new 关键字

new 在执行时会做四件事情：

  1. 在内存中创建一个新的空对象。
  2. 让 this 指向这个新的对象。
  3. 执行构造函数里面的代码，给这个新对象添加属性和方法。
  4. 返回这个新对象（所以构造函数里面不需要return）

Date 对象

Date 对象和 Math 对象不一样，它是一个构造函数，使用前要实例化。

如果Date()不写参数，就返回当前时间；如果Date()里面写参数，就返回括号里的时间。例如日期格式字符串为‘2019-5-1’，可以写成new Date(‘2019-5-1’) 或者 new Date(‘2019/5/1’)

JavaScript 中的垃圾回收

计时器setInterval()

使用setInterval方法时，每一次启动都需要对setInterval方法返回的值做一个判断，判断是否是空值，若不是空值，则要停止定时器并将值设为空，再重新启动，如果不进行判断并赋值，有可能会造成计时器循环调用，在同等的时间内同时执行调用的代码，并会随着代码的运行时间增加而增加，导致功能无法实现，甚至占用过多资源而卡死奔溃。

数组对象

创建数组对象有两种方式：字面量方式、new Array()

检测是否为数组：instanceof 运算符，可以判断一个对象是否属于某种类型；Array.isArray() 用于判断一个对象是否为数组

添加/删除数组元素的方法：push(参数1…)、pop()、unshift(参数1…)、shift()

数组排序：reverse() 颠倒数组中元素的顺序，无参数，会改变原来的数组；sort() 对数组元素进行排序，会改变原来的数组。

数组索引方法：indexOf(searchvalue, fromindex) 查找给定元素的第一个索引，lastIndexOf() 数组中的最后一个元素的索引。两者的返回值都是：如果存在则返回索引号，若不存在则返回 -1；

数组转换为字符串：toString() 把数组转换为字符串，逗号分隔每一项；join(‘分隔符’) 参数可选，指定要使用的分隔符，若省略则默认使用逗号作为分隔符。

concat() 连接多个数组，不改变原数组。

slice(start, end) 返回一个新的数组，包含从 start 到 end（不包括）的元素，不改变原数组。

splice(开始位置, 删除个数, item1…)，该方法向数组中添加/删除项目，返回 不包含被删除项目 的新数组，会影响原数组。

字符串变量

字符串的不可变，指的是里面的值不可变，虽然看上去可以改变内容，但其实是地址变了，内存中新开辟了一个内存空

charAt(index) 返回指定位置的字符；charCodeAt(index) 获取指定位置字符的ASCII码

字符串操作方法：concat() 拼接字符串；substr(start, length) 截取字符串，从start位置开始；slice(start, end)，end (取不到)；substring(start, end) 基本和 slice 一样，但不接受负数。

replace(被替换的字符串，要替换为的字符串) 。

split() 切分字符串为数组，返回 一个新数组。

toUpperCase() 转换为大写；toLoewrCase() 转换为小写。
